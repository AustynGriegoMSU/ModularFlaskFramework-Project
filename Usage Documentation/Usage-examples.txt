# Modular Flask Framework - Usage Examples

This document provides practical examples of how to build different types of applications using the modular Flask framework.

## Quick Start Examples

### Example 1: Simple Blog Site
```python
from library import create_app

# Create a basic blog with user accounts
app = create_app(
    modules=['blog', 'auth', 'database', 'dashboard'],
    config={
        'THEME': 'light-professional',
        'DASHBOARD_TYPE': 'blog'
    },
    site_name='My Tech Blog'
)

# What you get:
# - User registration and login
# - Blog post creation and management
# - SQLite database for persistence
# - Blog-focused dashboard with analytics
# - Professional light theme
```

## Advanced Configuration Examples

### Custom Database Location
```python
import os

app = create_app(
    modules=['blog', 'auth', 'database', 'dashboard'],
    config={
        'DATABASE_PATH': os.path.join('data', 'my_custom.db'),
        'THEME': 'dark-modern',
        'DASHBOARD_TYPE': 'blog'
    }
)
```

### Environment-Based Configuration
```python
import os

# Read configuration from environment variables
app_type = os.environ.get('APP_TYPE', 'blog')
theme = os.environ.get('THEME', 'light-professional')

if app_type == 'community':
    modules = ['blog', 'chat', 'auth', 'database', 'dashboard']
    dashboard = 'chat'
elif app_type == 'portfolio':
    modules = ['blog', 'dashboard', 'auth', 'database']  
    dashboard = 'blog'
else:
    modules = ['blog', 'dashboard', 'auth', 'database']
    dashboard = 'blog'

app = create_app(
    modules=modules,
    config={
        'THEME': theme,
        'DASHBOARD_TYPE': dashboard
    }
)
```

## Module Combinations Guide

### Minimal Configurations
```python
# Static website only
app = create_app(modules=['main'])

# Simple dashboard
app = create_app(modules=['dashboard'])

# Blog without user accounts (demo data only)
app = create_app(modules=['blog'])
```

### Recommended Combinations
```python
# Personal blog
modules = ['blog', 'auth', 'database', 'dashboard']

# Community platform  
modules = ['blog', 'chat', 'auth', 'database', 'dashboard']

# Business website
modules = ['dashboard', 'blog', 'auth', 'database']
```

## Dashboard Types Explained

### Blog Dashboard (`'blog'`)
- Content creation tools
- Writing workflow management
- Post analytics and performance
- Content pipeline (drafts, published, scheduled)
- Best for: Content creators, bloggers, writers

### Chat Dashboard (`'chat'`)
- Active conversation overview
- Online friend status
- Real-time messaging interface
- Communication analytics
- Best for: Community sites, gaming platforms

### Default Dashboard (`'default'`)
- General purpose overview
- System status and stats
- Quick actions for all modules
- Best for: Admin panels, general apps

## Real-World Deployment Examples

### Local Development
```python
# app.py
from library import create_app

app = create_app(
    modules=['blog', 'chat', 'auth', 'database', 'dashboard'],
    config={
        'THEME': 'dark-modern',
        'DEBUG': True,
        'DATABASE_PATH': 'instance/dev.db',
        'DASHBOARD_TYPE': 'default'
    },
    site_name='Dev Site'
)

if __name__ == '__main__':
    app.run(debug=True, host='0.0.0.0', port=5000)
```

### Production Configuration
```python
import os

app = create_app(
    modules=['blog', 'chat', 'auth', 'database', 'dashboard'],
    config={
        'SECRET_KEY': os.environ.get('SECRET_KEY', 'production-secret-key'),
        'DEBUG': False,
        'DATABASE_PATH': os.environ.get('DATABASE_URL', 'instance/production.db'),
        'THEME': os.environ.get('THEME', 'light-professional'),
        'DASHBOARD_TYPE': os.environ.get('DASHBOARD_TYPE', 'default')
    },
    site_name=os.environ.get('SITE_NAME', 'My Platform')
)
```

## Troubleshooting Common Scenarios

### Missing Dependencies
```python
# This will auto-add 'auth' and 'database' for you
app = create_app(modules=['blog', 'dashboard'])

# Console output:
# üìã Requested: ['blog', 'dashboard']
# ‚úÖ Loading: ['database', 'auth', 'blog', 'dashboard'] 
# ‚ö†Ô∏è  Auto-added dependency 'database' required by 'blog'
# ‚ö†Ô∏è  Auto-added dependency 'auth' required by 'blog'
```

### Template Safety
```python
# Templates use safe_url_for to prevent broken links
# If 'blog' module isn't loaded, links return '#' instead of breaking
<a href="{{ safe_url_for('blog_home') }}">Blog</a>
```

### Database Location Issues
```python
# Ensure database is saved to correct location
import os

# Project root instance folder (recommended)
database_path = os.path.join(os.getcwd(), 'instance', 'app.db')

app = create_app(
    modules=['database'],
    config={'DATABASE_PATH': database_path}
)
```

## Performance Tips

### Minimize Modules for Speed
```python
# Only load what you need
# Faster startup, smaller memory footprint
app = create_app(modules=['blog', 'auth', 'dashboard'])  # Essential only

# vs loading everything
app = create_app(modules=['blog', 'chat', 'auth', 'database', 'dashboard'])
```

### Theme Caching
```python
# Use cache buster for theme updates in production
app = create_app(
    config={
        'THEME': 'dark-modern',
        'CACHE_BUSTER': '2.0'  # Increment when CSS changes
    }
)
```
