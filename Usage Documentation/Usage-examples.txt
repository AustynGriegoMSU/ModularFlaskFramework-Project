# Modular Flask Framework - Usage Examples
This document provides practical examples of how to build different types of applications using the modular Flask framework.

## Quick Start Examples

### Example 1: Simple Blog Site
```python
from library import create_app

# Create a basic blog with user accounts
app = create_app(
    modules=['blog', 'auth', 'database', 'dashboard'],
    config={
        'THEME': 'light-professional',
        'DASHBOARD_TYPE': 'blog'
    },
    site_name='My Tech Blog'
)

# What you get:
# - User registration and login
# - Blog post creation and management
# - SQLite database for persistence
# - Blog-focused dashboard with analytics
# - Professional light theme
```

## Dashboard Types Explained

### Blog Dashboard (`'blog'`)
- Content creation tools
- Writing workflow management
- Post analytics and performance
- Content pipeline (drafts, published, scheduled)
- Best for: Content creators, bloggers, writers

### Chat Dashboard (`'chat'`)
- Active conversation overview
- Online friend status
- Real-time messaging interface
- Communication analytics
- Best for: Community sites, gaming platforms

### Default Dashboard (`'default'`)
- General purpose overview
- System status and stats
- Quick actions for all modules
- Best for: Admin panels, general apps
