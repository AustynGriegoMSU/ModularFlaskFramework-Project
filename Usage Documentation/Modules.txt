TECHNICAL NOTES:
---------------
- All modules use safe_url_for() for cross-module URL building
- Templates automatically adapt based on available_modules context
- Database connections are managed through context processors
- Authentication state is globally available when auth module is loaded
- Modules can extend base templates with their own nav_links blocks

REQUIREMENTS FOR USAGE:
----------------------
- Flask 3.1.2+
- Werkzeug 3.1.0+
- bcrypt 4.1.2 (for auth module)
- python-dotenv 1.0.0

HOW TO USE MODULES:
------------------
In app.py you change the requested modules in the modules section:
    User can modify based on 6 available modules:
        modules=['chat','blog','dashboard','database', 'auth', 'main']
If something is removed that is required by another module, dependency protection will add it automatically.
