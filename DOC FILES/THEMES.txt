HOW TO USE THEMES:
-----------------

METHOD 1 - Set Default Theme in App Creation:
```python
from library import create_app

# Light professional (default)
app = create_app(modules=['dashboard'], config={'THEME': 'light-professional'})

# Dark modern
app = create_app(modules=['dashboard'], config={'THEME': 'dark-modern'})

# Cyberpunk neon
app = create_app(modules=['dashboard'], config={'THEME': 'cyberpunk-neon'})
```

METHOD 2 - Update App Config:
```python
app = create_app(modules=['dashboard'])
app.config['THEME'] = 'cyberpunk-neon'  # Switch to cyberpunk theme
```

METHOD 3 - Environment Variable (Future):
```bash
export FLASK_THEME=dark-modern
```

CUSTOMIZING THEMES:
------------------

All themes use CSS custom properties (variables) for easy customization:

```css
:root {
    --primary-color: #your-color;
    --bg-primary: #your-background;
    --text-primary: #your-text-color;
}
```

You can:
1. Copy an existing theme file
2. Modify the CSS variables
3. Save with a new name
4. Use in your app config

CREATING NEW THEMES:
-------------------

1. Copy one of the existing theme files:
   ```
   cp static/themes/light-professional.css static/themes/my-theme.css
   ```

2. Modify CSS variables and styles

3. Use in your app:
   ```python
   app = create_app(config={'THEME': 'my-theme'})
   ```
