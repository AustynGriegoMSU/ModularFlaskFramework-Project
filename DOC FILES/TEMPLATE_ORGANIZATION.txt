TEMPLATE INHERITANCE:
--------------------

All module templates still extend base.html:
```html
{% extends "base.html" %}
{% block content %}
  <!-- Module-specific content -->
{% endblock %}
```

The base.html template remains in the root templates/ folder as it's shared across all modules.

ADDING NEW MODULES:
------------------

When creating new modules:

1. Create new template folder: templates/[module_name]/
2. Add module templates to the folder
3. Update routes to use: render_template('[module_name]/template.html')
4. Templates still extend base.html normally

Example for 'blog' module:
```python
# In routes.py
return render_template('blog/post_list.html')
return render_template('blog/post_detail.html')
return render_template('blog/post_create.html')
```
