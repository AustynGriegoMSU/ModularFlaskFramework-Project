Get Started in Seconds

Create a full-featured Flask app with just a few lines of code:

# app.py
from library import create_app

app = create_app(modules=[
    'main', 'auth', 'database', 'dashboard'
])

if __name__ == '__main__':
    app.run(debug=True)
